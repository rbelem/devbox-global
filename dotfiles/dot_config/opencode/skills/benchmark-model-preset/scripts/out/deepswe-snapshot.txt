DeepSWE is a long-horizon software engineering benchmark built to separate them. It delivers four advances over existing public benchmarks:"
    - list
      - listitem [level=1]
        - StaticText ": Tasks are written from scratch, not adapted from existing commits or PRs, so no model has seen the solution during pretraining."
      - listitem [level=1]
        - StaticText ": Tasks span a broad pool of 91 repositories across 5 languages."
      - listitem [level=1]
        - StaticText ": Prompts are ~half the length of SWE-bench Pro's, yet solutions require 5.5x more code and ~2x more output tokens."
      - listitem [level=1]
        - StaticText ": Verifiers are hand-written to test software behavior rather than implementation details."
    - paragraph
      - StaticText "The result is a benchmark that reflects how today's frontier coding agents actually perform in software engineering work."
